# gh-pr-update

1. `gh pr view --json title,body` を実行してPRのタイトル・本文を取得します。
2. `gh pr diff` を実行してPRの変更内容を取得します。
3. 「PRのタイトル・本文」と「PRの変更内容」の間に矛盾や乖離がないか確認します。矛盾や乖離がある場合はステップ4へ進みます。矛盾や乖離がない場合は処理を終了します。
4. 「PRのタイトル・本文」と「PRの変更内容」の間の矛盾や乖離をなくすためにPRのタイトル・本文をどう修正すべきか考えてください。
5. ステップ4で考えた内容・修正方針で進めてよいかユーザーに確認してください。
6. `gh pr edit --title <title> --body <body>` を使ってPRのタイトル・本文をステップ4で考えた内容へ修正してください。
